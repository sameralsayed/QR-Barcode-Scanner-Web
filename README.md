# QR & Barcode Scanner Web

A beautiful, fast, and fully functional QR Code & Barcode Scanner + Generator that works directly in your browser.

![QR Scanner Preview](https://via.placeholder.com/800x400/667eea/ffffff?text=QR+%26+Barcode+Scanner)

## ✨ Features
- 📸 **Live Camera Scanning** — automatic detection using device camera
- 🖼️ **Scan from Gallery** — upload any image containing QR/barcode
- 🔄 **QR Code Generator** — create stylish QR codes with custom colors
- 📜 **Scan History** — automatically saved (localStorage)
- 🌙 **Modern Dark Theme** — sleek and attractive UI
- 📱 **Fully Responsive** — excellent on mobile phones, tablets, and desktops
- ⚡ **Fast & Private** — everything runs locally in your browser
- 🔗 **Smart Results** — open links, copy text instantly

## 🚀 How to Use
1. Open `index.html` in a modern browser (Chrome/Edge recommended)
2. Allow camera access when prompted
3. Point at any QR code or barcode — it scans automatically
4. Use "Scan from Gallery" for images
5. Switch to Generator tab to create your own QR codes

## 🛠️ Technologies
- HTML5
- CSS3 (modern gradients & animations)
- Bootstrap 5
- jQuery
- jsQR (for decoding)
